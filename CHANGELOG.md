# Changelog

## 1.2.1

- Added PostToolUse hook event mapping (thinking state)

## 1.2.0

- New background scenes (park evening, park night, Japanese house, cyberpunk city)

- Draggable cats — drag any cat to reposition it freely
- Snooze reminders — plays a sound at configurable intervals while waiting for permission

- Background scene selection (bg1: forest path, bg2: park garden)

- Walking entrance animation from left or right depending on spot
- Click "done!" cat to dismiss it (cat goes to sleep)
- Static sleeping cat (no animation while sleeping)
- Multiple Claude session cats with directional entrance

## 0.1.0

- Initial release
- Cat animations for editor activity (typing, saving, errors, sleep)
- Claude Code session tracking with per-session cats
